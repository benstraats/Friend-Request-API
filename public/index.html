Friend request API Documentation:



USERS
./users - GET - find - returns the list of users (used for search)
./users/x - GET - get - returns the specific users info (Not Allowed)
./users - POST - create - Creates a new user specific to the payload (email, name, and password as JSON)
./users or ./users/x - PUT - update - Updates the user(s) info (Not Allowed)
./users or ./users/x - PATCH - patch - Update a user field (Not Allowed)
./users or ./users/x - DELETE - delete - Delete a specific user (Not Allowed)

AUTHENTICATION
./authentication - POST - create - Gets a authentication for the user given in the payload if successful (strategy, email, password as JSON)


Must be authenticated and send the accessToken with each of the below calls. Access token holds the info of the current user

FRIENDS
./friends - GET - find - Returns the list of users current user is friend with
./friends/x - GET - get - Returns true if current user and target are friends
./friends - POST - create - Creates a friend bond between current user and target only if target requested to be friends with current user
./friends or ./friends/x - PUT - update - Not Allowed
./friends or ./friends/x - PATCH - patch - Not Allowed
./friends or ./friends/x - DELETE - delete - Will delete the friendship (Not Allowed)

REQUESTS
./requests - GET - find - Gets all the requests (both as requestee and requester) for user
./requests/x - GET - get - Returns status of currUser and target (no request, already friends, currUser is requestee, currUser is requester)
./requests - POST - create - Creates a request where currUser is requester and target is requestee
./requests or ./requests/x - PUT - update - Updates the request (Not Allowed)
./requests or ./requests/x - PATCH - patch - Updates the request (Not Allowed)
./requests or ./requests/x - DELETE - delete - Deletes the request (Not Allowed)

PROFILE
./profile - GET - find - Gets the profile for the current user
./profile/x - GET - get - Gets the profile for the specified user
./profile - POST - create - Creates the profile for the user
./profile or ./profile/x - PUT - update - Updates the whole profile for the currUser
./profile or ./profile/x - PATCH - patch - Updates a specific row for the currUser
./profile or ./profile/x - DELETE - delete - Deletes the profile of the user (Not Allowed)
