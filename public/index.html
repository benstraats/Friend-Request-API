Section Path REST Call Feather Call Description Returns Authentication
Users ./users GET find Searches the user database Returns a list of user emails and names Yes
Users ./users/x GET get Gets the info on a specific user Returns Error, this action isn't allowed Yes
Users ./users POST create Creates a new user specific to the payload Returns response 200 upon success No
Users ./users/x PUT update Updates a given user Returns Error, this action isn't allowed Yes
Users ./users/x PATCH patch Fixes a field for a specific user Returns Error, this action isn't allowed Yes
Users ./users/x DELETE delete Deletes a specific user Returns Error, this action isn't allowed Yes
Authentication ./authentication POST create Creates an accessToken for given user info Returns a hashed accessToken No
Friends ./friends GET find Gets current users list of friends A list of users emails and names Yes
Friends ./friends/x GET get Gets if current user and target are friends Returns true if they are. Yes
Friends ./friends POST create Creates a friend pair between current user and target (only if valid request exists) The validated friend pair Yes
Friends ./friends/x PUT update Updates the friend pair. Returns Error, this action isn't allowed Yes
Friends ./friends/x PATCH patch Changes an attribute for the friend pair Returns Error, this action isn't allowed Yes
Friends ./friends/x DELETE delete Deletes the friend pair Returns Error, this action isn't allowed Yes
Requests ./requests GET find Gets all the requests current user is apart of Returns a list of requester and requestee pairs for current user Yes
Requests ./requests/x GET get Gets the status of current user and target request Returns no status, friends, curr user is requestee or requester Yes
Requests ./requests POST create Creates a request to the target if not already friends or requested Returns the new request Yes
Requests ./requests/x PUT update Updates the target request Returns Error, this action isn't allowed Yes
Requests ./requests/x PATCH patch Fixes an attribute for the target request Returns Error, this action isn't allowed Yes
Requests ./requests/x DELETE delete Deletes the target request Returns Error, this action isn't allowed Yes
Profile ./profile GET find Gets all the profiles the user is friends with Returns a list of info on all profiles Yes
Profile ./profile/x GET get Gets a specific users profile Returns just the users profile Yes
Profile ./profile POST create Creates a profile for the current user Returns the profile Yes
Profile ./profile/x PUT update Updates the current users profile Returns the updated profile Yes
Profile ./profile/x PATCH patch Updates a part of the current users profile Returns the updated part of the users profile Yes
Profile ./profile/x DELETE delete Deletes the users profile Returns Error, this action isn't allowed Yes
